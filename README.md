# SkillSwap – Plateforme d'Offres d'Emploi (Angular)

## Description

SkillSwap est une application web développée avec Angular qui permet aux utilisateurs de se connecter et de consulter des offres d'emploi dans le domaine des technologies.

Ce projet démontre l'utilisation des concepts fondamentaux d’Angular tels que les composants, le routage, l’utilisation d’API REST et la création d’une interface utilisateur moderne.

Les données (utilisateurs et offres d’emploi) sont récupérées dynamiquement à partir d’une API externe.

---

## Fonctionnalités

* Connexion d’un utilisateur via une API
* Récupération des données depuis une API REST
* Affichage dynamique des offres d'emploi
* Navigation entre les pages avec Angular Router
* Interface utilisateur moderne et responsive

---

## Technologies utilisées

* Angular
* TypeScript
* HTML
* CSS
* API REST (jsonplaceholder)

---

## Comment lancer le projet

### 1. Installer les dépendances

npm install

### 2. Lancer l'application Angular

ng serve

### 3. Ouvrir l'application dans le navigateur

http://localhost:4200

## Structure du projet

src/app/components

* login → Page de connexion
* register → Page d'inscription
* jobs → Page des offres d'emploi
* job-detail → Page de détail d'une offre

src/app/services

* job.service → Gestion des appels API
